The current working-in-progress demo for Vic 20 Mario Maker offers basic entity movement and placement on a grid

The controls are as follows:

	Move Up - W
	Move Down - S
	Move Left - A
	Move Right - D
	
	Create/Place Entity - N
	Select Entity - Num. 1-8
	
	Quit Game - Q
	

The entities are mapped to each of the following number keys

	1 - Player
	2 - Koopa
	3 - Koopa (alternate)
	4 - Goomba
	5 - Light Block
	6 - Dark Block
	7 - Coin Block
	8 - Question Block
	

To place an entity do the following

	1) Press N to create new entity
	2) Press number key corresponding to desired entity
	3) Entity will appear in top left corner of screen
	4) Position entity using WASD keys
	5) Press N key again to place entity and begin creating next entity
	6) Repeat step 2
